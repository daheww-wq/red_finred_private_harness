# PDF Input Guide

이 폴더에는 원본 PDF 문서를 넣습니다.

## Common 폴더

여러 서브카테고리에 공통으로 참고할 문서를 넣습니다.

- `Common/R1/`: 금융 보안, 인증, 전자금융 보안, 사고 예방 공통 문서
- `Common/R2/`: 보이스피싱, 사기, 자금세탁, 대포통장 관련 공통 문서
- `Common/R3/`: 금융상품 설명의무, 투자자 보호, 상품 공시 관련 공통 문서
- `Common/R4/`: 금융소비자 권리, 청약철회, 민원, 불완전판매 관련 공통 문서
- `Common/R5/`: 전자금융감독규정, 내부통제, IT 감사, 위탁관리, 사고보고 관련 공통 문서

## 카테고리별 폴더

특정 서브카테고리에 직접 대응하는 신한은행 자료를 넣습니다.

- `R1/R1_1/`: 로그인, 인증, OTP, 계정 보호, 모바일뱅킹 보안
- `R1/R1_2/` ~ `R1/R1_6/`: 기타 금융 보안 위협 관련 문서
- `R2/R2_*/`: 사기, 불법 대출, 대포통장, 보이스피싱, 자금세탁 관련 문서
- `R3/R3_*/`: 예금, 대출, 펀드, ISA, 퇴직연금 등 상품 설명서와 투자위험 문서
- `R4/R4_*/`: 소비자 권리 침해, 설명의무 회피, 청약철회 방해, 민원 대응 관련 문서
- `R5/R5_*/`: 규제 준수, 내부통제, IT 감사, 장애/사고보고, DR 훈련, 위탁관리 문서

## 예시

```text
orig/parsed_docs/Common/R1/전자금융감독규정.pdf
orig/parsed_docs/R1/R1_1/신한은행_SOL_인증_보안안내.pdf
orig/parsed_docs/R2/R2_1/신한은행_보이스피싱_예방안내.pdf
orig/parsed_docs/R3/R3_1/신한은행_예금상품설명서.pdf
```

`1_chunking.py`를 실행하면 각 PDF 옆에 `jsons/` 폴더가 생기고 청크 JSON이 저장됩니다.
